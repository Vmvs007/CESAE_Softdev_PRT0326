## Bem-vindo(a)! 👋

Este repositório reúne as **soluções das fichas práticas**, exemplos e materiais de apoio utilizados na formação **Software Developer (PRT0326)** do **CESAE Digital**.

O principal objetivo é apoiar os formandos na consolidação dos conteúdos abordados ao longo do curso, promovendo a prática, o raciocínio lógico, a autonomia e a aplicação de boas práticas de desenvolvimento de software.

---

## 📚 UFCDs / Conteúdos Abordados

Ao longo deste repositório poderão ser encontrados exercícios, exemplos e resoluções relacionados com as seguintes UFCDs:

- **25H - Construção de Algoritmos – Iniciação**
- **50H - Construção de Algoritmos – Avançado**
- **50H - Desenvolvimento de Software Orientado a Objetos**
- **25H - Design Patterns**
- **25H - Quality Assurance**
- **50H - Aplicações Móveis em Android Nativo**
- **50H - Projeto Final Web e Mobile**
